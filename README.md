# protobuf-sample
###The most simple implementation of protocol buffers in practice.
#####Protobuf (or protocol buffers) is a language-neutral, platform-neutral, extensible way of serializing structured data for use in communications protocols. It's firstly developed by Google. But its support for javascript is implemented mostly by github developers.
-----------------------------------------
```bash
$ git clone https://github.com/emranbm/protobuf-sample.git
$ cd client/
$ npm install
$ cd ../server
$ npm install
$ npm start
```

and in other terminal:
```bash
$ cd client/
$ npm start
```
